Add attribute dynamically

Elemnt where data-id attr will change

Toggle class green and blue

Elemnt where class value will change

Js code for the same

const instance = MiniJs.create({
    dynamicAttr: "login",
    attrValue: "",
    className: "green"
})
document.getElementById('btn').addEventListener("click", (e) => {
    instance.dynamicAttr = instance.attrValue;
})
document.getElementById('btn1').addEventListener("click", (e) => {
    instance.className = instance.className == 'green' ? "blue" : "green";
})
